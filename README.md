# Concrete & Reinforcement Quantity Calculator

## Overview

A web-based civil engineering calculator developed using Python to estimate concrete quantities, reinforcement quantities, and preliminary material costs for residential construction projects.

The application helps civil engineers and quantity surveyors perform faster quantity takeoff, improve calculation accuracy, and simplify project documentation.

## Features

- Concrete quantity estimation
- Reinforcement quantity estimation
- Material cost estimation
- Project information documentation
- Quantity takeoff support
- User-friendly web interface

## Technologies

- Python
- GitHub
- AI-assisted development

## Purpose

This project was developed to improve engineering calculation efficiency and support construction professionals during project planning and execution.
